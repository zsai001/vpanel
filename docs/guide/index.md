# 简介

VPanel 是一个新一代的智能服务器运维管理平台，旨在为运维人员和开发者提供一个简单、高效、安全的服务器管理解决方案。

## 什么是 VPanel？

VPanel 是一个开源的、企业级的服务器管理面板，它提供了一套完整的服务器运维工具，包括：

- 🐳 **Docker 容器管理** - 完整的容器生命周期管理
- 🌐 **Nginx 管理** - 站点配置、SSL 证书、反向代理
- 🗄️ **数据库管理** - 多数据库支持，备份恢复
- 📁 **文件管理** - Web 文件管理器
- 💻 **终端** - Web SSH 终端
- ⏰ **计划任务** - Cron 任务管理
- 🔒 **防火墙** - 安全规则管理
- 🔌 **插件系统** - 可扩展的插件架构

## 核心特性

### 插件化架构

VPanel 采用插件化架构设计，核心功能精简高效，扩展功能通过插件实现。你可以：

- 从官方插件市场安装功能插件
- 开发自己的私有插件
- 与团队分享自定义插件

### 多节点管理

支持通过 Agent 管理多台服务器：

```
┌─────────────┐
│   VPanel    │
│   Server    │
└──────┬──────┘
       │
   ┌───┴───┐
   │       │
┌──▼──┐ ┌──▼──┐
│Agent│ │Agent│ ...
└─────┘ └─────┘
```

### 现代化 UI

基于 React 构建的现代化用户界面：

- 响应式设计
- 深色/浅色主题
- 国际化支持

## 技术栈

| 组件 | 技术 |
|------|------|
| 后端 | Go (Gin/Fiber) |
| 前端 | React + TypeScript + Vite |
| UI | TailwindCSS |
| 状态管理 | Zustand |
| 数据库 | SQLite / PostgreSQL |
| 缓存 | Redis (可选) |
